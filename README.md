# Дизассемблер 

Простой дизассемблер для **16-битных DOS-программ**, написанный на чистом **FASM** (ассемблер). Имеет графический интерфейс (Windows), но также поддерживает запуск из командной строки.

## О проекте

Инструмент для дизассемблирования 16-битных `.COM` и `.EXE` файлов. Проект представляет собой учебный прототип — ничего сверхсложного, но вполне рабочий дизассемблер для изучения структуры машинных кодов x86.

##  Возможности

- Дизассемблирование 16-битных исполняемых файлов (DOS)
- **Два режима работы:**
  - **Графический режим** — оконное приложение Windows с удобным интерфейсом
  - **Терминальный режим** — запуск из командной строки для автоматизации
- Преобразование машинного кода в мнемоники ассемблера
- Полностью на ассемблере (FASM)
- Компактный размер, быстрая работа

## Технологии

- **FASM** (Flat Assembler) — 100% кода
- Платформа: Windows (GUI + консоль)
- Целевая архитектура: x86 (16-bit real mode)

### Запуск

**Графический режим:**
```bash
disassembler.exe
```
Откроется окно, где можно выбрать файл для дизассемблирования.

**Терминальный режим:**
```bash
disassembler.exe program.com dis.txt
```
или
```bash
disassembler.exe program.exe dis.txt
```
## Статус

Прототип — базовая функциональность работает. Проект сделан скорее для себя, чтобы разобраться в теме дизассемблирования и низкоуровневого программирования.
